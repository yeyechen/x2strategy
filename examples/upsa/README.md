This directory is the UPSA paper2code example, generated by Copilot GPT-5.4.

| File | Purpose |

## Notes

- The UPSA example is a research/backtest contract over factor return series.
- It is not directly broker-connected paper trading because the strategy input is factor returns and the output is factor-portfolio weights.
- The copied input files are `input/P10_Kelly_Malamud_Pourmohammadi_Trojani_2025_NBER.pdf`, `input/jkp_factors_wide.csv`, `input/jkp_factors_long.csv`, `input/upsa_weights.csv`, and `input/sample_instruction.md`.
- Some legacy example files may remain for regression tests.
